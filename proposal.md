# KO Final Project Proposal

## Repository
Link to the public repository can be found [HERE](https://github.com/xanthophyll101/xanthO_final_project.git)

## Description
This project will allow users to design their own crochet pattern and export it as an image file.

## Features
- Create a simple pattern with limited stitches
	- Users will have a bank of 2-4 types of common crochet stitches (single crochet, double crochet, etc). They will click what type of stitch they want, and then click the canvas to paste it.
- Rotating the stitch once placed
	- Users will have a 'rotate' tool that will allow them to rotate the stitches to be more readable.
- Exporting the finished image
	- Once users are done with their design, they'll be a button where they can export the image as a JPEG.

 - Common crochet stitches. Minimum stitches implemented will be chain, single crochet, and double crochet.
 ![Common crochet stitches. Minimum stitches implemented will be chain, single crochet, and double crochet.](https://makersnook.com/wp-content/uploads/2020/05/crochet-chart-key.jpg)


## Challenges
- Figuring out the MSPaint style selection system (once tool selected -> tool does thing)
- How to paste an image onto a canvas with user input, and how to rotate pasted image
- How to change the color of the pasted stitch image with user input (color wheel, basic colors)
- How the button will export the image (how to choose the path directory, how to handle user input, etc)

## Outcomes
Ideal Outcome:
- Users will be able to make a garment/scarf design with the tools provided, as well as change the color of the stitch

Minimal Viable Outcome:
- Users will be able to make a basic design (ex. granny square), but without the color choices.

## Milestones

- Week 1
  1. Canvas and other visual aspects can be seen, but not interacted with.
  2. One tool can be interacted with OR one stitch can be pasted onto the canvas

- Week 2
  1. All tools are implemented
  2. Export button is implemented

- Week N (Final)
  1. Added color choices to the pasted stitch image
  2. Final polishing for the user interface is completed
