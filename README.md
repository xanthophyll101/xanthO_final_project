## Description
My final project for Programming for Digital Arts is the Crochet Pattern Maker!

Essentially, you can make your own crochet patterns with this program. It includes the chain stitch, single crochet, and double crochet pattern symbols.

Clicking on the crochet pattern symbols in the top-right will spawn the corresponding symbol. Move it where you want on the canvas, and then spawn another one. The rest you already know if you crochet at all: repeat, repeat, repeat!

# Controls:
    Edit/Select Mode - [/] respectively
    WASD/Arrow Keys - Move the most recently spawned symbol
    O - Rotate counterclockwise
    P - Rotate clockwise
    Backspace - Delete the most recently spawned symbol

The requirements.txt file lists the pip libraries you will need to run the Python program. The src directory contains the Python file (project.py) as well as all of the images used in the GUI. Proposal.md contains the original pitch for this project, and README.md is the file you're reading right now!

For my GUI design, I went with an old, weathered envelope type look. I used free images for all the GUI designs, although I did alter some with Photoshop to fit the look I had in mind. 

If I had more time, first and foremost I would add more crochet stitches. I would also make the control scheme a bit more intuitive. For example, I tried making Left Alt + Arrow Keys the controls for rotating the symbol but worried I wouldn't have enough time to make the GUI look how I wanted. There's a lot more critiques I could say, I'm an artist after all, but those were the main gripes.

This was my first big programming project, so I hope you like it! Happy crafting!
